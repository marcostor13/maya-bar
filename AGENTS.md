# Guía de Agentes del Proyecto BAR

Este proyecto implementa eficiencias de tokens para Claude y Gemini basadas en `drona23/claude-token-efficient`.
Los LLMs deben omitir saludos, disculpas, justificaciones extensas, y enfocarse en dar la respuesta exacta y directa a la instrucción.

## Reglas (.cursor/rules/)
- Se enfocarán en convenciones limpias (SOLID) tanto para Angular como para NestJS.
- **NestJS**: Uso de DI, Módulos, Servicios, DTOs y Guards.
- **Angular**: Arquitectura de componentes Standalone, manejo de Signals, Control Flow de Angular 20.

## Skills (.cursor/skills/)
Se implementarán habilidades modulares que le darán contexto al agente sobre:
1. `nestjs-auth`: Estructura para JWT y Mongoose en el backend.
2. `angular-auth`: Patrones funcionales para el frontend.
