UI & Design Guidelines (BAR Project):
- Estética: App Móvil B2C/B2B2C Premium. Muy limpia y apetitosa.
- UI Metrics: Abundante aire visual (padding de 24px en tarjetas). Bordes extremos (radius-pill 9999px para botones, radius-lg 24px para cards).
- CSS Variables: Obligatorio usar styles.scss. Colores principales: --color-brand #E11D48, fondos --color-brand-light #FFF0F3.
- Tipografía: Poppins (--font-heading) para titles/H1 y botones. Inter (--font-base) para textos normales.
- Sombras y Animaciones: Sombras difusas y amplias (--shadow-lg). Interacciones fluidas (hover eleva card -4px). 
- Botones y Formularios: .btn por defecto (14px). Usar .btn-sm en tablas/cards, .btn-lg para CTAs, .btn-icon para íconos. Para formularios usar SIEMPRE las clases .input, .select, .textarea (diseño pill-shaped unificado).
- Íconos: Siempre usar `lucide-angular` (lucide-icon). No usar caracteres de texto ni emojis para la UI.
